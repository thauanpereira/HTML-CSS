Itálico / Ênfase

Texto Marcado

Podemos criar um texto marcado usando a TAG <MARK>.

Texto Deletado

Podemos marcar um texto excluido para mostrar que ele não deve ser considerado, usando a TAG <del>.

Texto Inserido

Da mesma maneira podemos marcar um texto que foi inserido usando a TAG <ins>.

Texto Sobreescrito

Para inserir palavras do tipo x5-3 usamos a TAG <sup>.

Texto Subescrito

E para escrever palavras do tipo H2O usamos a TAG <sub>.
