Citações Simples

Como diria Thiago Finch: Perfeito é melhor do que feito.

Para fazer essa citação basta utilizar a TAG <q>.

Citações Completas

Umas vez ouvi de uma pessoa:

Se não nasceu em berço de ouro, seja imbatível para dormir em uma cama de diamantes.

Para fazer esse tipo de citação basta usar a TAG <blockquote>.

Abreviações

Quando mais novo eu costumava jogar muito CA com meus amigos.

Podemos inserir significado de abreviações com a TAG <abbr>.
